# Hosting Panel  'Guten'

## About Project

Hosting Panel  'Guten' based on Laravel framework

## Install

Run commands

`
composer create-project chirukinbb/hosting-panel-guten path --repository-url=https://github.com/chirukinbb/hosting-panel-guten
`

and

`
artisan migrate
`

after

`
artisan db:seed
`
